BatSSharing_Collect.bat - Data W Y
If you double-click this shortcut ...
1. Divide the Data folder into three.
2. Move one to drive W.
3. Move one to drive Y.

BatSSharing_Distribute.bat - Data 2 W Y
If you double-click this shortcut ...
1. Move the file on drive W to Data_shr.
2. Move the file on drive Y to Data_shr.
3. Reconstruct the folder Data_shr.



BatSSharing_Collect.bat - Data W Y
このショートカットをダブルクリックすると・・・
1.Dataフォルダーを３つに分散します。
2.１つをドライブWに移動します。
3.１つをドライブYに移動します。

BatSSharing_Distribute.bat - Data 2 W Y
このショートカットをダブルクリックすると・・・
1.ドライブWのファイルをData_shrに移動します。
2.ドライブYのファイルをData_shrに移動します。
3.フォルダーData_shrを再構築します。
